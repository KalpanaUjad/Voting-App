Routes

1. User Authentication
   /signup : POST - create a new user account.
   /login : POST - log in to an existing account

2. voting
   /candidates : GET - Get the list of candidates.
   /vote/:candidateId: POST - vote for a specific candidate.

3. Vote counts
   /vote/counts : GET - Get the lis of candidates soted by their vote counts.

4. User Profiles
   /profile : GET - Get the Users profile information
   /profile/ password : PUT - Change the users's password

5. Admin Candidate Management
   /candidate : POST - create a new candidate 
   /candidate/:candidateID : PUT - Update an existing candidate
   /candidates/:candidateID : DELETE - Delete a candidate from the list.
